CS:GO MM SERVER PICKER


DOWNLOAD

Avaible on Windows & Linux : https://github.com/Jyben/csgo-mm-server-picker/releases/latest

DESCRIPTION

This application allows you to select the Steam servers on which you want to play for Counter Strike : Global Offensive.
Administrator rights are required to block the IP addresses of the servers. For security reasons, requests for rights are systematic with each click on the "GO" and "Reset" buttons.
The IP addresses are automatically updated without any action on my part and the application is updated automatically.
The application is developed with the Electron framework and its source code is available on my Github.
I accept pull requests only for bug fixes.
